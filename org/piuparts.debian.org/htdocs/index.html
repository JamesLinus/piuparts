piatti.debian.org / piuparts.cs.helsinki.fi

This machine is piatti.debian.org, generously donated by HP and hosted at piuparts.cs.helsinki.fi by the University of Helsinki, CS department.

piatti runs piuparts, currently doing package installation and removal tests in sid (using http://ftp.fi.debian.org/debian/dists/sid/main/binary-amd64/Packages.bz2). Upgrade tests are also planned, as well as better report pages and PTS integration. Join #debian-qa if you want to help.
