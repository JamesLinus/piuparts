This machine is piatti.debian.org, generously donated by HP and hosted at piuparts.cs.helsinki.fi by the University of Helsinki, CS department.

piatti runs piuparts, currently the setup is still being polished. Better reports and stats as well as PTS integration is planned. Join #debian-qa if you want to help.

Test cases:
  • package installation and removal in sid
  • package installation and removal in squeeze
  • package installation in lenny, upgrade to squeeze and removal in squeeze

These pages are updated every 30 minutes.

More information about the setup is available at wiki.debian.org/piuparts.
